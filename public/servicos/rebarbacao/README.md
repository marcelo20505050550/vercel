## 📸 PASSO 1: Adicionar Fotos Antes/Depois

### Caminho completo:
```
C:\Backup\Aplicativos\bvboaventura\public\servicos\rebarbacao\antes-depois\
```

### Nomes dos arquivos:
- ✅ `antes-1.jpg` - Primeira foto ANTES do processo
- ✅ `depois-1.jpg` - Primeira foto DEPOIS do processo
- ⭐ `antes-2.jpg` - Segunda foto ANTES (opcional)
- ⭐ `depois-2.jpg` - Segunda foto DEPOIS (opcional)

### Formatos aceitos:
- `.jpg` ou `.jpeg`
- `.png`
- `.webp` (recomendado para web)

### Dicas:
- Resolução recomendada: 1200x900px ou maior
- Tamanho máximo: 2MB por foto
- Use nomes exatamente como mostrado acima

---

## 🎥 PASSO 2: Adicionar Vídeo (quando estiver pronto)

### Caminho completo:
```
C:\Backup\Aplicativos\bvboaventura\public\servicos\rebarbacao\videos\
```

### Nome do arquivo:
- ✅ `processo-rebarbacao.mp4`

### Formatos aceitos:
- `.mp4` (recomendado)
- `.webm`

### Dicas:
- Resolução recomendada: 1920x1080px (Full HD)
- Tamanho máximo: 50MB
- Duração recomendada: 30-60 segundos
